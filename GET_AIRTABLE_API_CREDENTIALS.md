# How to Get Your Airtable API Credentials

## Step 1: Get Your Base ID

1. **Go back to your Airtable base** (the table you're working in)
   - You should see "Joe's Used Harleys Inventory" at the top

2. **Click on "Help"** (usually in the top right, or look for a question mark icon)

3. **Click "API documentation"** or **"API"**

   OR

   **Alternative method:**
   - Go directly to: https://airtable.com/api
   - You'll see a list of all your bases
   - Click on **"Joe's Used Harleys Inventory"**

4. **You'll see your Base ID** at the top of the page
   - It looks like: `appXXXXXXXXXXXXXX`
   - It might say "Base ID:" or just show it in the URL
   - **Copy this entire ID**

---

## Step 2: Get Your API Key

1. **On the same API documentation page**, scroll down to the **"Authentication"** section

2. **Click "Show API key"** or **"Generate API key"**

3. **Copy your API Key**
   - It looks like: `patXXXXXXXXXXXXXX`
   - **Copy this entire key**

   **OR if you don't see it:**

4. **Go to your account settings:**
   - Click your profile icon (top right, usually a circle with your initial)
   - Click **"Account"** or **"Settings"**
   - Look for **"Developer"** or **"API"** section
   - Click **"Generate API key"** or **"Show API key"**
   - Copy it

---

## Step 3: Get Your Table Name

1. **Look at your Airtable base**
2. **Check the left sidebar** - you should see "Table 1" (or whatever you named it)
3. **That's your table name!** Usually it's just **"Table 1"**

---

## Step 4: What You Should Have

You need these 3 things:

1. **Base ID**: `appXXXXXXXXXXXXXX` (starts with "app")
2. **API Key**: `patXXXXXXXXXXXXXX` (starts with "pat")
3. **Table Name**: `Table 1` (or whatever your table is called)
