# Internal Linking Strategy for Joe's Used Harleys

## Overview

A comprehensive internal linking strategy distributes link equity, improves crawlability, and enhances user experience. This document outlines 25 strategic internal links with anchor texts, placement recommendations, and SEO reasoning.

---

## 1. Homepage → Inventory Page

**Anchor Text:** "View Complete Inventory" or "Browse Used Harleys for Sale"

**Placement:** Hero section CTA, main navigation, footer

**SEO Reasoning:** 
- Distributes homepage authority to inventory page
- Signals inventory page importance to search engines
- Improves crawlability of product listings
- Targets "used harleys for sale" keyword

## Best Practices

1. **Use Descriptive Anchor Text:** Match anchor text to target keywords naturally
2. **Maintain 3-Click Rule:** Users should reach any page within 3 clicks
3. **Create Topic Clusters:** Group related content with internal links
4. **Use Contextual Links:** Place links within relevant content
5. **Avoid Over-Optimization:** Vary anchor text to appear natural
6. **Monitor Link Equity:** Ensure important pages receive more links
7. **Update Regularly:** Add new links as content is created
